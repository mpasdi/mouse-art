# mouse-art

一款 改变 鼠标点击和移动效果的插件。给你提供丰富的特效选择

## 使用

### 1. 安装
```
npm install mouse-art
```

### 2. 使用
```markdown
import 'mouse-art/dist/index.css'
import mouseArt from 'mouse-art'

mouseArt({
  effectType: 'love_rise'
})

effectType: 代表类型。目前可选类型如下
- text_hidden
- love_rise
- circle_enlarge
- circle_cross_explosive
```
